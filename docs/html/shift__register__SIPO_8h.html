◆ shift_register_SIPO_enable()

void shift_register_SIPO_enable ( ShiftRegisterSIPO *  sr,
bool  value 
)

Set output enable.

The output enable pin on the SN74HC595 is active low, so the GPIO will be set to the complement of the boolean entered here; i.e. false will set the GPIO high.

Note
The enable pin needs to be set by shift_register_SIPO_set_enable_pin() prior to calling this function
Parameters
srshift register
valuetrue enables the output, false disables it
